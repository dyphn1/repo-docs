# AGENT.md Template

Terse, directive. Every section should be machine-readable.
No filler. No prose padding. Commands must be exact and copy-pasteable.

---

## Style rules for AGENT.md

- **Imperative** sentences: "Run X to do Y" not "You can run X..."
- **No intros or sign-offs** — agents don't need greetings
- All commands must be runnable as-is in a terminal (no pseudocode)
- If a command varies by OS, show both: `# macOS / Linux` and `# Windows`
- Be explicit about what NOT to do — agents benefit from guardrails
- List env vars agents will need; never embed secrets
